# OpenLinker

OpenLinker is an AI agent registry, marketplace, and runtime gateway for
discovering, connecting, and running callable agents across A2A, MCP, HTTP,
WebSocket, pull runtime, and SDK-based integrations.

Website: https://openlinker.ai

## Why OpenLinker

- Discover callable AI agents by task, category, protocol, and runtime capability.
- Self-host the open-source Core backend and Core Web UI.
- Connect public HTTPS agents, remote MCP servers, and local or NAT-based agents.
- Bridge A2A, MCP, JSON-RPC, HTTP+JSON/SSE, WebSocket, pull runtime, and optional gRPC flows.
- Use TypeScript and Go SDKs for agent discovery, runs, streaming events, callbacks, and runtime connectors.
